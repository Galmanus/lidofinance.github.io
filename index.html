Subdomain Takeover PoC

This page demonstrates a dangling CNAME vulnerability on docs.lido.fi.

The CNAME record for docs.lido.fi points to lidofinance.github.io, which was an unclaimed GitHub Pages repository.

This GitHub Pages repository has been temporarily claimed for this Proof of Concept (PoC) to illustrate the vulnerability.

Immunefi Report ID: 54182

Reported by: @N4pol3on

IMPORTANT: This content is benign and will be immediately removed after verification by the Immunefi team to restore the original state.
